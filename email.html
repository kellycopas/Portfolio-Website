EMAIL DEVELOPMENT & CSS ANIMATION

Working as a Digital Developer involved the development and testing of emails for clients such as Samsung, St John's Ambulance, and Opus Energy. This has helped to develop HTML, CSS, and responsive design skills.

It was necessary to continue to test new email techniques including CSS animations whilst also providing fallbacks for email clients with less support. A particular task was using CSS transitions to create an image scrolling effect so that the image appears to be moving from left to right. However, not all email clients support transitions and so it also was necessary to set the fallback to be a GIF.

Another example was using CSS transitions to add a fade-in effect to the main content of an email to emphasise the launch of the new Samsung S9 phone.
