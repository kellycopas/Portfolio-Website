EMAIL DEVELOPMENT & CSS ANIMATION

As part of my University degree, I undertook a year long work placement in a digital marketing agency where I was working as a Digital Developer. The role involved building and testing emails and web banners for clients such as Samsung, St John's Ambulance, and Opus Energy. This has helped to develop HTML, CSS, and responsive design skills.

To retain customer engagement, it was necessary to continue to test new email techniques, including CSS animations, whilst also adding fallbacks for email clients with less support. A particular task I worked on was to use CSS transitions to create an image scrolling effect so that the image appears to be moving from left to right (above). However, not all email clients support CSS transitions and so it was also necessary to set a fallback to be a GIF. Therefore, attention to detail and testing were important to ensure emails display correctly on multiple browsers and devices.

Another example that I built was to use CSS transitions to add a fade-in effect to the main content of an email to emphasise the launch of the new Samsung S9 phone.
